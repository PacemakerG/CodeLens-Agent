**1、增加版本号**

增加pyproject.toml文件中的version

**2、更新install.sh**

更新install.sh中whl文件的版本号。

**3、打包**

```bash
uv build --wheel
# 生成 dist/deep_ai_analysis-<version>-py3-none-any.whl
```

如需先准备本地开发环境：

```bash
uv sync --extra dev
source .venv/bin/activate
```

激活后 CLI 命令写法保持不变，例如：

```bash
deep-ai-analysis --help
```

**4、上传whl包和install.sh**

安装脚本和whl文件放在[测试环境s3 ad-dqe-public/ai-coding-analysis](https://s3plus.mws-test.sankuai.com/bucket-detail/beijing/com.sankuai.wmadbizcharge.bizad.aegis/ad-dqe-public/objects/ai-coding-analysis%2F?project_id=)
